## üìÇ The 30-Day Roadmap

### Phase 1: Reconnaissance & Enumeration (Days 1-10)
* **Active & Passive Recon:** OSINT, Google Dorking, and WHOIS.
* **Scanning:** Master Nmap, service identification, and vulnerability scanning.
* **Web Enumeration:** Directory brute-forcing and hidden file discovery.

### Phase 2: Exploitation & Vulnerabilities (Days 11-20)
* **Web Attacks:** SQL Injection, XSS, and IDOR.
* **Network Attacks:** Exploiting SMB, FTP, and SSH vulnerabilities.
* **Metasploit Framework:** Payload generation and listener configuration.

### Phase 3: Post-Exploitation & Reporting (Days 21-30)
* **Privilege Escalation:** Moving from user to root/admin in Linux and Windows.
* **Data Exfiltration:** Safely moving data out of a target network.
* **Reporting:** How to document findings for a professional client.
